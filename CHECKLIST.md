### Что реализовано:
```

✅ 8 REST API эндпоинтов
✅ Фильтрация товаров по 7+ параметрам
✅ Автоматическая инициализация БД при старте
✅ 18 реалистичных тестовых товаров
✅ Генерация URL товаров и рыночных цен
✅ Валидация параметров (Pydantic)
✅ Асинхронная обработка (asyncio)
✅ Фейковые User-Agent для имитации браузера
```

### API эндпоинты:
```
GET /api/items                   # Товары с фильтрацией
GET /api/filters/brands          # Список брендов
GET /api/filters/categories      # Список категорий
GET /api/filters/countries       # Список стран
GET /api/filters/sizes           # Список размеров
GET /api/filters/price-range     # Min/Max цены
GET /api/filters/marketplaces    # Список маркетплейсов
GET /api/health                  # Проверка здоровья
```

## 🚀 Как запустить?

### Способ 1: Автоматический запуск (PowerShell)
```powershell
.\start.ps1
```

### Способ 2: Автоматический запуск (CMD)
```cmd
start.cmd
```

### Способ 3: Автоматический запуск (Bash/Linux)
```bash
bash start.sh
```

### Способ 4: Ручной запуск
```bash
# Терминал 1
cd backend && pip install -r requirements.txt && python main.py

# Терминал 2
cd frontend && npm install && npm run dev
```

**Затем откройте:** `http://localhost:3000`
