🚀 КАК ЗАПУСТИТЬ? (ВЫБЕРИТЕ ОДИН ВАРИАНТ):

Вариант 1️⃣: PowerShell (Windows)
  .\start.ps1

Вариант 2️⃣: CMD (Windows)
  start.cmd

Вариант 3️⃣: Bash (macOS/Linux)
  bash start.sh

Вариант 4️⃣: Ручной запуск
  Терминал 1:
    cd backend && pip install -r requirements.txt && python main.py
  
  Терминал 2:
    cd frontend && npm install && npm run dev

Затем откройте в браузере:
  http://localhost:3000
